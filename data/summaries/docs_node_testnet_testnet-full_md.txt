To join the Ergo testnet, download the latest Ergo protocol reference client and launch it with the command `java -jar -Xmx4G ergo-*.jar --testnet -c testnet.conf`. A minimal `testnet.conf` file is provided. The node will be available at `localhost:9052/panel`, and the Swagger user interface is available at `localhost:9052/swagger`. If you want to help mine the testnet, you can follow the steps outlined in the CPU Mining document.
