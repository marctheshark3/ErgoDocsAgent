ErgoPi is a guide for setting up an Ergo full node on a Raspberry Pi. It covers the prerequisites, preparation steps, and configuration options for running an Ergo node on a Raspberry Pi. The guide includes instructions for increasing the swap space, optimizing the setup for the Raspberry Pi's limitations, and provides a quick start script as well as a step-by-step manual setup guide. It also discusses light mode configuration and various configuration options that can be tweaked in the ergo.conf file. The document includes links to additional resources and tutorials for running an Ergo full node on a Raspberry Pi.
