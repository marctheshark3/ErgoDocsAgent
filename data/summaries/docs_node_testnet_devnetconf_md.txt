This document describes the configuration settings for the Ergo node in the Development Network (devnet). It covers the main sections and fields of the configuration file, including network type, chain configuration, wallet configuration, and Scorex configuration. The network type is set to "devnet" for development purposes. The chain configuration includes parameters such as protocol version, address prefix, initial difficulty, and voting mechanisms. The wallet configuration sets the directory for the wallet's secret storage. The Scorex configuration covers network settings like magic bytes, bind address, node name, and known peers, as well as the REST API configuration for the API key hash.
