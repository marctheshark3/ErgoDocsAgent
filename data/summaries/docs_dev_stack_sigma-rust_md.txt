Sigma Rust is a Rust implementation of the ErgoScript cryptocurrency scripting language. It provides a set of crates, including ergo-lib, ergotree-interpreter, ergotree-ir, ergoscript-compiler, and sigma-ser, which expose various features related to the Ergo blockchain, such as chain types, JSON serialization, box selection, transaction building, and signing. The project also offers language bindings for Wasm, JavaScript/TypeScript, Swift (iOS), Java/JVM, C, Go, and Ruby/Python, allowing developers to interact with the Ergo blockchain using their preferred programming languages. The document provides usage examples in Rust and TypeScript/JavaScript, showcasing how the Sigma Rust library is being used in various Ergo-related projects.
