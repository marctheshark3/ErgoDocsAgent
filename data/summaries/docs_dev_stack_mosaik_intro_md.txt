The document introduces Ergo Mosaik, a UI system for Ergo dApps. Ergo is a smart contract platform that requires both on-chain and off-chain code. Mosaik is a JSON-based markup language served via a REST API, intended to be used by Ergo dApps. It provides view elements and actions typically needed for blockchain dApps, allowing separation of UI and off-chain code. Mosaik supports various programming languages and wallet integrations. The document provides a hands-on tutorial to get started with Mosaik, demonstrating its features through a sample application.
