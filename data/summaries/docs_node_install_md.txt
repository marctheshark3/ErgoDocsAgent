The Ergo Reference Client (Node) is the core software that connects to the Ergo P2P network, validates transactions and blocks, and maintains a copy of the entire blockchain. Running a node is crucial for the network's decentralization and security. The document outlines the installation options and resources for the Ergo reference client, including minimum requirements, pre-requisites, and different modes of operation such as Pruned Full Node and Full Archival Node. It also provides information on alternative options like integrated wallets that offer a built-in node installation.
