ErgoScript is a language used to write contracts for the Ergo blockchain. ErgoScript contracts can be compiled to ErgoTrees, serialized, and stored in UTXOs. The language has features like Scala-like syntax, high-order functions, static typing, and predefined types like Box, SigmaProp, and GroupElement. The document provides an overview of the language features, data types, and examples of using the language with the Ergo Appkit library.
