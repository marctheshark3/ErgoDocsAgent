Minotaur Wallet is the inaugural multi-platform wallet for the Ergo ecosystem, offering features like wallet generation, mnemonic phrase support, read-only functionality, token and NFT management, transaction previews, embedded dApp support, and cross-platform availability. Upcoming features include dynamic dApp integration, browser extensions, multilingual support, and mempool transaction monitoring. The wallet can be built from source, with platform-specific instructions provided for Android, iOS, and desktop. Contributions to the developer are appreciated.
