This document provides a guide on developing simple pay-to-script (P2S) applications using ErgoScript. It introduces the P2S address compiler available at wallet.plutomonkey.com/p2s and provides examples of applications that can be created, such as Wrapped ERG, Crowdfunding, and 3-out-of-5 Threshold Signature. The document explains the key features and functionality of these example applications, allowing users to understand the capabilities of P2S applications built with ErgoScript.
