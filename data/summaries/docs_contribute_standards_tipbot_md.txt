Ergotipper is a bot that allows users to tip other members with Erg or any of the supported native tokens. Users can set up a wallet, fund it, and start tipping. The bot supports multiple platforms including Reddit, Discord, Telegram, and Twitter. To get started, users can follow the instructions for their respective platform. The bot also allows users to restore their wallet using the same seed phrase and use the same tip wallet across all platforms. The document provides a list of supported tokens and instructions on how to add new tokens. It also includes a list of commands for interacting with the bot on each platform.
