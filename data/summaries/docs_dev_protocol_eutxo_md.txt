The UTXO model provides several benefits, including privacy, scalability, interoperability, and predictable transaction costs. Ergo leverages these advantages and extends the UTXO model with eUTXO, which allows smart contracts to use UTXOs as read-only data inputs without modifying them. This enables parallel computation and non-custodial atomic swaps. The eUTXO model also supports Turing-complete smart contracts through a multi-stage UTXO approach.
